# **App Name**: Tran Quang Khai Web

## Core Features:

- Multi-Page Structure: Implement multi-page architecture as requested with Next.js, enabling structured navigation.
- Core Page Development: Develop dedicated pages for the introduction, activities, garden, documents, contact information, and events.
- Interactive Wishes Form: Create a form for sending wishes, triggering a congratulatory animation upon submission, rendered with client-side JavaScript. Firebase is not used for MVP.
- Friendly URLs: Implement a URL structure that's both clean and user-friendly using Next.js routing.
- Branded Color Palette: Incorporate branding with blue (#1791db), red (#d64545), green (#58c65e), and orange (#e85d18) color schemes.

## Style Guidelines:

- Primary color: Saturated blue (#1791db), reminiscent of the sky, suggesting dependability and clarity.
- Background color: Desaturated light blue (#E3F2FD), maintaining the calming effect of blue, but light and unobtrusive.
- Accent color: Complementary vivid teal (#00A39E) to highlight key interactive elements
- Headline font: 'Instrument Sans' (sans-serif) for a modern aesthetic.
- Body font: 'Jost' (sans-serif) for readability.
- Font Awesome to render interface icons where needed.
- Subtle gradient animations on main headers.
